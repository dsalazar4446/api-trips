# Trip API

## Description
APi desarrollada con el proposito de registrar y cosultar viajes

## Dependencias

- [Git](https://git-scm.com/) Control Version
- [Docker](https://docs.docker.com/get-docker/) container and image manager
- [NodeJs](https://nodejs.org/) JavaScript runtime
- [NPM](https://www.npmjs.com/) package manager
- [Nest](https://nestjs.com/) framework


#### Clonar repositorio

**HTTPS:**

```bash
$ git clone https://github.com/dsalazar4446/api-trips.git
```

**SSH:**

```bash
$ git clone git@github.com:dsalazar4446/api-trips.git
```

## Installation

```bash
$ npm install
```

## Seeding
```bash
# crear registros
npm run seed

# refrescar registros
npm run seed:refresh
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
# Docker 
```bash
npm run docker
```

## Swagger
```
http:localhost:3000/api-docs/
```
